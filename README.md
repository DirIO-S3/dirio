# dirio-s3.github.io source (gh-pages branch)

Source for the DirIO marketing/download page, served by GitHub Pages directly
from this branch. Deliberately self-contained and independent of `main` —
edit it directly on this branch (see `pages:worktree` / `pages:preview` Task
targets on `main` for local editing).

`dirio_logo.svg` is a vendored copy of `main`'s `assets/dirio_logo.svg`. If
the logo changes on `main`, re-copy it here manually — nothing syncs it
automatically.
